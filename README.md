# نظام إدارة مكتبات ومراقبة الكتب (كتالوج، إعارة، إرجاع)
==============================================

## Overview & Project Purpose
---------------

نظام إدارة مكتبات ومراقبة الكتب هو مشروع يهدف إلى توفير حل شامل لإدارة المكتبات، بما في ذلك إدارة الكتب، الإعارة، والإرجاع. يهدف النظام إلى تسهيل عمليات إدارة المكتبات وتحسين كفاءة العمل.

## Project Structure Mapping
-------------------------

المشروع يتكون من المكونات التالية:
* `app`: تطبيق الويب الرئيسي
* `database`: قاعدة البيانات
* `docker`: ملفات تكوين الدوكر
* `requirements`: متطلبات التطبيق

## Step-by-Step Instructions on Running the Environment
------------------------------------------------

لتشغيل البيئة باستخدام `docker-compose up`، اتبع الخطوات التالية:
1. قم بتثبيت `docker` و `docker-compose` على جهازك.
2. قم بتشغيل الأمر `docker-compose up -d` في المجلد الرئيسي للمشروع.
3. انتظر حتى يتم تشغيل جميع الخدمات.
4. قم بفتح متصفح الويب وادخل إلى الرابط `http://localhost:8000`.

## Listing of Modules, Tables, and Roles
--------------------------------------

### Tables
* `books`: جدول الكتب
* `borrowing`: جدول الإعارة

* `users`: جدول المستخدمين

### Roles
* `admin`: مدير النظام
* `librarian`: أمين المكتبة
